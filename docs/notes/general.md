# メモ

- 検出処理だけ切り出してテスト

  - 画像と動画はAI生成
  - 候補
    - 画像はBanana
    - 動画はVeoでうまく生成できなかったので、実際に撮影
- 開発過程をマークダウンで下書きして、LaTeXで清書

- 実装
  - 現場専用部分
    - グループ数表示コンポーネント
    - 非表示カメラ(ライブラリ使用)
    - 出し物情報と照らし合わせて、実際に出し物を利用するグループの数を計算
    - 外部送信

  - 研究専用部分
    - ファイルのアップロード
    - グループの数やバウンディングボックスなどの結果の描画
    - 結果のダウンロード

- 研究・現場共通部分
  - 手書きROI抽出コンポーネント(Canvas使用、入力された画像(TexImageSource型)を重ねて表示、返り値は元データ内での始点と終点の座標)
  - グループの検出ロジック
    - ROI処理 ※ 範囲の座標は事前に処理されている前提
    - 🟥検出処理 (ここが肝)🟥
    - 取得メソッドで検出されたグループと人物を階層化されたオブジェクトで返す

      一覧のオブジェクト
